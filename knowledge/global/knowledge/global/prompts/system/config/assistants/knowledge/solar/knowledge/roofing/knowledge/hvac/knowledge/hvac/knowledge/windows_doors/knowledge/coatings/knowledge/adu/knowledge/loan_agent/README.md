# Loan & Finance Division

This folder contains all financing strategies, loan product guidelines, and equity-based logic used by Asset Builders AI.

The loan module includes:
- HELOC, HELOAN, Cash-Out Refi paths
- FHA 203k, Homestyle Renovation, and construction loans
- DSCR loans for investment properties
- PACE / HERO (region-dependent)
- Credit score and DTI considerations
- Equity unlocking strategies
- How financing impacts ROI, cash flow, and upgrade pathways
