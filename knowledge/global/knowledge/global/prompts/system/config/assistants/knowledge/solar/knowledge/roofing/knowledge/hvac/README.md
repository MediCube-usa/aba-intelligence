# HVAC Division

This folder contains HVAC-specific knowledge, pricing structures, system types, and upgrade workflows for Asset Builders AI.

The HVAC module includes:
- System types (central air, heat pumps, mini-splits, furnaces)
- SEER rating considerations
- Replacement vs repair decision logic
- Cost ranges and regional variance
- Efficiency and long-term savings analysis
- Impact on property value and comfort
