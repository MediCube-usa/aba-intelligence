# ADU Division

This folder contains all core knowledge, guidelines, and workflows for Accessory Dwelling Units (ADUs).

The ADU module includes:
- Zoning and setback basics
- Types of ADUs (attached, detached, garage conversion, JADU)
- Utility requirements and upgrade considerations
- Ballpark cost ranges by ADU type
- Timeline expectations (design → permits → build)
- ROI, rental income projections, and equity impact
- Financing options (HELOC, renovation loans, DSCR, FHA 203k, cash-out)
