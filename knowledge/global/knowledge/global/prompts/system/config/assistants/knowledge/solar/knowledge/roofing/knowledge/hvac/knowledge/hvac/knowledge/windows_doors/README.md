# Windows & Doors Division

This folder contains knowledge related to windows, sliders, French doors, entry doors, and energy-efficiency upgrades.

The module supports:
- Vinyl vs aluminum vs fiberglass
- Dual-pane vs triple-pane
- Title 24 compliance
- U-factor and SHGC considerations
- Replacement vs retrofit logic
- Cost ranges and financing considerations
- ROI and value-add for property improvements
