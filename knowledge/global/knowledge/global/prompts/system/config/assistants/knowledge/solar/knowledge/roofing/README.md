# Roofing Division

This folder will contain roofing standards, best practices, pricing structures, material options, and upgrade rules used by Asset Builders AI. 

The roofing module supports:
- tear-off vs overlay logic  
- material selection (comp, tile, metal, flat roof systems)  
- cost ranges  
- permitting considerations  
- insurance and storm-damage scenarios  
- value-add and ROI reasoning
